CSS Types in react

External CSS or Sass file (Global, cause problems)
CSS Modules (Component) [One external file per component]
Other CSS frameworks
Tailwindcss
Material UI
Chakra UIM
Mantine

Both create react app and create vite comes pre-installed with css modules
In order to use it, for eg in Product.jsx file
create a file named Product.module.css
import styles from "./Product.module.css"
className = {styles.nav}
Dont use element selector, instead give class names to the element in the jsx file and using that apply css in module.css file.
